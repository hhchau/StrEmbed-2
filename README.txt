StrEmbed-2

1/ hypercube.pl - Generate 2^n hypercube lattice, output to STDOUT
2/ em2.pl - Embed a STEP AP214 file onto a 2^5 hypercube lattice, Perl/Tk GUI
3/ chase.pl - Rewrite a STEP AP214 with a new assembly structure, output pp.STEP

Usage:

1/ At a command line prompt,
   type "perl hypercube.pl n" where n is an interger greater than or equals to 0
2/ At a File Manager window, double click em2.pl
3/ At a File Manager window, double click chase.pl
